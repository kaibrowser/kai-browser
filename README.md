# kai-browser
A Python-based browser where you can create custom extensions using natural language. Describe what you want, and AI builds it.

## Installation

### Download and Run
```bash
git clone https://github.com/exfo999/kai-browser.git
cd kai-browser/dist
mkdir -p modules
chmod +x kai_browser
./kai_browser
```
